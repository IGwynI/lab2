# Задание 1

## Задача 1

### Текст задачи

Получить список из минимальных цифр натуральных чисел, содержащихся в
исходном списке

### Алгоритм решения

Ввод количества элементов; 
Формирование списка из элементов, вводимых пользователем; 
Поиск минимальной цифры в каждом числе списка, с помощью List.map применяется функция min_digit ко всем элементам списка, функция отбрасывает цифры числа справа и если очередная цифра меньше чем текущий минимум, то эта цифра становится новым минимумом; 
Вывод результата пользователю

# Задание 2

## Задача 1

### Текст задачи

Найти количество элементов списка, в которых встречается заданная цифра.

### Алгоритм решения
Ввод количества элементов; 
Формирование списка из элементов, вводимых пользователем; 
Запрос искомой цифры; 
Проверка наличия и счет цифры в числе в каждом числе исходного списка с помощью List.fold, acc является аккумулирующим значением, которое увеличивается если функция containsDigit возвращает истину, функция containsDigit отбрасывает цифры числа справа пока не найдет искомую цифру, либо пока число не станет равным 0; 
Вывод результата пользователю
